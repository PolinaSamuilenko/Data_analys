# 1 модель - Жилой одноэтажный дом 
В качестве первой модели  было выбрано одноэтажное жилое здание (файл ifc. был взят из интернета). При помощи П.О. Speckle и Power BI из файла  с расширением ifc. были cняты метрики и переведены в формат сsv., для дальнейшего статистического анализа в Orange. В power BI, кроме того, были созданы графики для оценки выборки, превалирующих групп и их количества в целом. В частности из модели, получена информация по стенам, окнам, крыше и полу, фурнитуре, дверям. Для каждой группы из модели получены такие метрические данные как:(heigth), ширина (width), рабочая площадь(net.area), расстояние от пола (SillHeight). 
Общая площадь здания - 214.7 м2.

# Окна
Всего 19 окон, по размеру делятся на 3 неравные группы: 1 окно размером 580*1180 расположенное в техническом помещении, 5 больших окон в пол 2180*2180, 13 окон 1010*1010. В данных по окнам удалось найти параметр .RoughWidth (приблизительная ширина), однако он присутствует только у окон в пол и у окна технического помещения. Т.о. у 68 процентов данных этого показателя нет. По графикам видно что есть четкое разделение данных на 3 группы. По результатам корреляции Пирсона получили что корреляция -1 у высоты и высоты от пола,что логично, и  + 1 у приблизительной ширины и ширины. При этом 
корреляции у высота от пола и других показателей всегда отрицательные. Рабочая площадь в большей степени зависит от высоты.

# Лестницы
В модели для  описания лестниц существует несколько типов файлов это IFCSTAIRFLIGHT, IFCSTAIR и IFCSLAB. В IFCSTAIRFLIGHT отдельно рассматриваются лестничные марши (4шт) и такие их характеристики как количество проступей, количество подступенков, их высота, ширина проступи и ширина марша. Элементы делятся на 2 группы по ширине марша,другие показатели одинаковы для каждого элемента.В IFCSTAIR каждый рассмотренный элемент представляет собой 2 лестничных марша, 2 промежуточные площадки и перила, то есть лестницы "целиком", из характеристик в IFCSTAIR нет ширины марша, но появилась характеристика начальный номер проступи. Нет деления на группы, оба элемента идентичны. В IFCSLAB заданы общая толщина и относительная высота промежуточных площадок. К тому же, для перекрытий (4 шт) числовых данных нет совсем, поэтому больше 60 процентов данных не получены.
